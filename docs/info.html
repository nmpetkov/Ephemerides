Ephemerides

Ephemerides is a Zikula module to enter and display past and current events. Provides a block displaying a bit of information (historical event, thought for the day, etc.) linked to the day's date, with daily roll-over, and incorporates an interface for adding, editing and maintaining ephemerids.

1. Categorization is enabled in Ephemeride block, so every instance of the block can associate with different category (categories).
2. Block content is cached for given interval, separate for every different instance of Ephemeride block. With this:
- improved performance of whole web site;
- change displaying content not per every click of the user, but in given time interval.
3. There are block display filters: by hour of the day, day of the week, day of the month, month.
4. There is a "status" column of ephemerids (Active/Inactive).
